TimeScale
含义:时间刻度入口；数量: 1
属性名称含义可用值默认值
dragMode鼠标拖拽方式 normal - 鼠标拖拽时自动横向伸缩
none - 鼠标不可拖拽		无


Segments
含义:"段"集合；数量:1;


Segment
含义:段；数量:1 - n;
属性名称含义可用值默认值
From该段的开始时间日期无
To该段的结束时间日期无
备注1：如果总共只有一个 "段"，则不需要指定 From、To (大多数情况下只有一个段);
备注2：如果有多个 "段"，第一段不需要指定 From、末段不需要指定 To;

tip鼠标划过时的文字显示掩码同上无
fontIndex字体数字, 树列表的字体集合 <Fonts> 中的顺序号0
align文字对齐left / center / rightcenter
备注：日期掩码宏的含义表(大小写敏感)
%x - 日期的普遍写法, 如 2014/03/22
%xd - 日期的普遍写法, 如 2014.03.22
%xm - 日期的普遍写法, 如 2014-03-22
%xE - 日期的普遍写法(英文), 如 03/22/2014
%c - 日期和时间, 如 2014/03/22 19:30:00
%cd - 日期和时间, 如 2014.03.22 19:30:00
%cm - 日期和时间, 如 2014-03-22 19:30:00
%cE - 日期和时间(英文), 如 03/22/2014 07:00:00 AM
%X - 时间, 如 19:30:00
%XE - 时间(英文), 如 07:00:00 AM
%Y - 4位数的年, 如 2014
%y - 2位数的年, 如 14
%k - 半年，1至2)
%kk - 半年, 如 上半年, 下半年
%kkE - 半年(英语), 如 1st, 2nd
%q - 季度, 1至4
%Q - 季度, 为一,二,三,四
%QE - 季度(英文), 为1st,2nd,3rd,4th
%m - 月份, 1至12
%mm - 月份, 01至12
%B - 月份的全称, 如 三月
%BE - 月份的全称(英文), 如 March
%b - 月份的简称, 如 一, 二
%bE - 月份的简称(英文), 如 Mar
%bme - 旬, 为1,2,3
%bmed - 旬, 为1, 11, 21
%BME - 旬, 为上旬, 中旬, 下旬
%BMEE - 旬, 为B, M, E
%A - 星期的全称, 如星期六
%AE - 星期的全称(英文), 如 Saturday
%a - 星期的缩写(一, 二...日)
%aE - 星期的缩写(英文), 如 Sat
%w - 数字表示的星期, 1至7 (1是星期天)
%d - 月中的某日, 1至31
%dd - 月中的某日, 01至31
%du - 离项目开始的天数, 1, 2, 3...
%dU - 离项目结束的天数, 90, 89, 88...
%H - 小时(24小时制，00至23)
%I - 小时(12小时制，01至12)
%M - 分钟(00至59)
%S - 秒钟(00至59)
%j - 相对于1月1日的日期差(001至366)
%p - 上午/下午
%pE - 上午/下午(英文), 如AM,PM
%Z - 和GMT(或UTC)时区的时差
